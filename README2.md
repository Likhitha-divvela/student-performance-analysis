🎓 Student Performance Analysis

📌 Project Overview

This project analyzes student performance based on various factors such as gender, parental education, lunch type, and test preparation course.

📊 Objectives

- Understand factors affecting student performance
- Perform data cleaning and feature engineering
- Visualize data using graphs
- Generate insights from data

🛠️ Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

📂 Dataset

The dataset contains student exam scores in:

- Math
- Reading
- Writing

And includes factors like:

- Gender
- Race/Ethnicity
- Parental level of education
- Lunch type
- Test preparation course

🔧 Data Cleaning

- Renamed columns to lowercase
- Removed spaces and replaced with underscores
- No missing values found
- No duplicate records found

⚙️ Feature Engineering

- Created "total_score" by summing all subject scores
- Created "average_score"
- Categorized students into performance levels (Good, Average, Poor)

📈 Exploratory Data Analysis

- Compared performance based on gender
- Analyzed impact of test preparation course
- Studied parental education influence
- Visualized score distributions
- Generated correlation heatmap

🔍 Key Insights

- Students who completed test preparation scored higher
- Female students performed better in reading and writing
- Math scores were slightly lower compared to other subjects
- Students with standard lunch performed better
- Reading and writing scores are highly correlated

🚀 Conclusion

This analysis helps understand key factors influencing student performance and highlights the importance of preparation and background factors.
